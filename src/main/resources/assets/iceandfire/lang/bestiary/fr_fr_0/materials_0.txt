Après avoir tué un dragon ou être
tombé sur son cadavre, il est
possible de le piller en
interagissant avec lui.






Les écailles de dragon sont l'un des
objets les plus courants que l'on
peut récupérer sur les bêtes mortes.
Elles peuvent être transformées en
armures très résistantes.





Les os de dragon constituent le
deuxième butin le plus courant.







Les os de dragon sont utilisés dans
de nombreuses recettes, mais surtout
pour la fabrication d'outils et
d'armes. Ces recettes impliquent
généralement l'utilisation d'os
flétris au lieu de bâtons. L'os de
flétrissure peut être récupéré sur
les squelettes noirs dans les bas-
fonds.
