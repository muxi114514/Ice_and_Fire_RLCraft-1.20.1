Un deuxième minerai magique associé
aux créatures surnaturelles est le
saphir.




Le minerai de saphir se trouve
aléatoirement dans les biomes
enneigés.



Les pépites d'or ou d'argent peuvent
être stockées en piles plaçables,
comme suit.











Le seul moyen d'obtenir plus
d'informations sur le monde magique
est de faire des recherches sur les
œuvres d'autres écrivains. Pour ce
faire, vous pouvez fabriquer un
lutrin de bestiaire.
